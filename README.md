# ImportCalc — China → Madagascar Import Simulator

A web app to simulate and manage clothing & accessories imports from China to Madagascar.

Built for personal use to track costs, customs fees, and client orders.

## Features

- 💱 Live CNY/Ariary exchange rate (auto-fetched)
- 🧮 Full cost calculator (transport, customs, VAT, misc fees)
- 👥 Client management with order tracking
- 📦 Per-item price suggestions with profit margins
- 🌙 Dark / Light mode
- 🔥 Real-time sync via Firebase (shared between devices)
- 📋 Documents guide for Madagascar customs
- 📖 Interactive import guide with step-by-step explanations

## How to use

Just open the link — no installation needed.

## Tech Stack

- Pure HTML / CSS / JavaScript
- Firebase Firestore (real-time database)
- Hosted on GitHub Pages
